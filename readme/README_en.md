## Install

```bash
# bash
pip install xy_pydev
```

## Start

```bash
# bash
xy_pydev -w clean
# 删除缓存

xy_pydev
# 出现引导，也可以按需提前设置参数

xy_pydev -m test_module_name -n test_name -s 1
```
